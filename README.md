This artpiece serves as our presentation for the vernissage on April 8th 2026.

The core idea was to showcase the art part of the "Algorithmic art" course, while still focusing on the algorithm and all that we've learned, and what's the best way to do so than using the code itself as the central focus of the piece.

### Echoes?

The echoes in Cascading Echoes refers to 2 features in the piece :

- Random comments and thoughts appearing randomly in the piece, sometimes even colliding with the code display, reflecting some of our emotions.

- Optionnaly, the background can showcase multiple smaller art pieces that all rely on tools (Algorithms, Data sources, etc.) we made use of throughout the semester, acting as **Echoes of the past**
